Trip Application (REST API & Admin Panel)

Features

  • Multi-user system where each user possesses an individual account with distinct roles and permissions.
  • Multi-Vendor system where each travel agenecy can manage their plans, trips and reservations.
  • Hotels listing.
  • Countries and cities management.
  • Reservation management.
  • Advanced datatables with filters, pagincation, column toggling, and smart search functionality.
