Document Management System

Features

  • Multi-user system where each user possesses an individual account with distinct roles and permissions.
  • Dynamic roles & permissions. Roles can be applied to departments.
  • The ability to add documents, assign them to specifics organization(s) or user(s).
  • Documents can have attachments, relations, margins and nested versions.
  • Each document has its own discussion to help employees comment and communicate with each other.
  • Advanced datatables with filters, pagincation, column toggling, and smart search functionality.
  • Additionally, the platform offers seamless email notifications, password resetting, email verification, and multi-language.
